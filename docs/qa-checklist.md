# Desktop Portal 发布前 QA 清单

版本：`0.1.0-beta`

## 安装和启动

- [ ] 从 `artifacts\DesktopPortal-0.1.0-beta-win-x64\DesktopPortal.exe` 启动。
- [ ] 再次启动同一个 exe，只保留一个进程和一个托盘图标。
- [ ] 关闭主窗口后，程序留在托盘。
- [ ] 托盘菜单可以打开设置。
- [ ] 托盘菜单可以退出程序。

## 规则管理

- [ ] 新增网页规则。
- [ ] 新增文件规则。
- [ ] 新增文件夹规则。
- [ ] 新增 exe 程序规则。
- [ ] 编辑规则后配置自动保存。
- [ ] 删除规则后配置自动保存。
- [ ] 右键菜单可以测试、编辑、复制、启用/禁用、删除规则。

## 快捷键

- [ ] F1-F12 可以录入并触发。
- [ ] 数字键可以录入并触发。
- [ ] Ctrl / Alt / Shift 组合键可以录入并触发。
- [ ] 鼠标中键可以录入并触发。
- [ ] 鼠标侧键可以录入并触发。
- [ ] 快捷键冲突会在状态列显示。
- [ ] 点击“健康检查”后，重复快捷键、无效快捷键和不存在的目标会在状态列显示。
- [ ] 暂停全部快捷键后不再触发规则。
- [ ] 恢复快捷键后规则重新生效。

## 目标执行

- [ ] URL 普通模式可以打开默认浏览器。
- [ ] URL 独立窗口模式优先使用 Chrome，其次 Edge。
- [ ] 文件已打开时优先切换到现有窗口。
- [ ] 文件未打开时可以打开文件。
- [ ] 文件夹已打开时优先切换到资源管理器窗口。
- [ ] exe 已运行时优先切换到现有窗口。
- [ ] 目标不存在时显示错误提示，不崩溃。

## 配置和日志

- [ ] 配置文件路径为 `%AppData%\DesktopPortal\config.json`。
- [ ] 删除配置文件后会自动创建默认配置。
- [ ] 保存配置后会在 `%AppData%\DesktopPortal\backups` 保留最近 5 份 `config.backup.*.json`。
- [ ] 损坏配置会备份为 `config.broken.*.json`。
- [ ] 日志路径为 `%AppData%\DesktopPortal\logs\app.log`。
- [ ] 日志轮转生效。
- [ ] 人为制造 UI 异常或后台异常时，异常信息会进入日志且程序不因日志写入失败而崩溃。

## 开机自启

- [ ] 勾选开机自启后写入 HKCU Run。
- [ ] 取消开机自启后删除 HKCU Run 值。
- [ ] 设置失败时只提示错误，不崩溃。
